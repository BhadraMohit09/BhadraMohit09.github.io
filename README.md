# 🚀 Bhadra Mohit – Full Stack Developer Portfolio

Welcome to the official repository of **Bhadra Mohit's Dynamic Portfolio**. This project showcases modern web development aesthetics, dynamic interactive elements, and responsive designs built with React, Vite, and Tailwind CSS.

## 🌟 Live Demo
Check out the live portfolio: **[https://bhadramohit.vercel.app](https://bhadramohit.vercel.app)**

---

## 🛠 Tech Stack
- **Frontend Framework:** React 19 + Vite
- **Styling:** Tailwind CSS + Vanilla CSS micro-animations
- **Animations & Graphics:** Framer Motion + tsParticles
- **PWA & SEO:** Vite PWA + React Helmet Async
- **Icons:** Lucide React & React Icons

---

## ⚙️ Local Development

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed.

### 1. Clone the repository
```bash
git clone https://github.com/BhadraMohit09/BhadraMohit09.github.io.git
cd BhadraMohit09.github.io
```

### 2. Install dependencies & run development server
```bash
# You can run directly from root:
npm run dev

# Or inside the mab-portfolio directory:
cd mab-portfolio
npm install
npm run dev
```

### 3. Build for Production
```bash
npm run build
```

## 📦 Deployment on Vercel
This repository is configured with a root `vercel.json` and `package.json` so that Vercel automatically builds and deploys the portfolio from any setting with zero configuration required!
