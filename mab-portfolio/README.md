# 🚀 Bhadra Mohit – Portfolio Application (`mab-portfolio`)

This is the primary frontend source code for **Bhadra Mohit's Full Stack Developer Portfolio**.

## 🌟 Features
- **Dynamic Animations:** Built with `framer-motion` and custom `tsparticles`.
- **Progressive Web App (PWA):** Offline capability, auto-update service workers, and app shortcuts.
- **Rich SEO:** Fully optimized meta tags, Open Graph tags, Twitter cards, and JSON-LD structured metadata.
- **Responsive & Modern UI:** Glassmorphism, smooth gradients, and interactive components.

## 📜 Scripts
- `npm run dev` - Start local development server
- `npm run build` - Build optimized production bundle
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint code inspection
